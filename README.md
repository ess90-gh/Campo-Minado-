📄 Descrição do Projeto – Campo Minado

O projeto consiste no desenvolvimento de um jogo interativo baseado no clássico Campo Minado, utilizando as tecnologias HTML, CSS e JavaScript. O objetivo principal foi criar uma aplicação funcional e intuitiva, permitindo ao usuário interagir com o jogo diretamente no navegador.

Inicialmente, foi construída a estrutura base do jogo, com a criação do tabuleiro, definição das bombas e implementação da lógica de funcionamento dos cliques. A partir disso, o projeto foi evoluindo com melhorias no código e na experiência do usuário.

⚙️ Funcionalidades Implementadas

Geração automática do tabuleiro
Distribuição aleatória de bombas 💣
Sistema de revelação de células
Contador de bombas ao redor
Sistema de bandeiras 🚩 (clique com botão direito)
Contador de tempo ⏱
Verificação de vitória e derrota
Reinício automático do jogo
Níveis de dificuldade (fácil, médio e difícil)
Inserção do nome do jogador 👤
Feedback visual e sonoro (animações e sons)

🎨 Melhorias e Alterações Realizadas

Durante o desenvolvimento, foram feitas diversas alterações em relação à versão inicial do jogo:

Ajuste no tamanho dos quadrados para melhor visualização
Reformulação do design, tornando o jogo mais moderno e interativo
Organização e melhoria na estrutura do código
Implementação de animações ao vencer o jogo
Adição de efeitos sonoros para interações
Criação de um sistema de níveis para variar a dificuldade
Inclusão de elementos visuais como ícones (💣🚩⏱)

Essas melhorias tiveram como objetivo proporcionar uma experiência mais agradável, dinâmica e acessível ao usuário.

🎯 Objetivo do Projeto

O projeto foi desenvolvido com fins educacionais, com o intuito de aplicar conceitos de lógica de programação, manipulação de DOM e organização de código, além de estimular a criatividade no desenvolvimento de interfaces.

🚀 Como Executar

Acesse o link do projeto no GitHub Pages
Insira seu nome (opcional)
Escolha o nível de dificuldade
Clique nos quadrados para jogar
Use o botão direito do mouse para marcar bandeiras 🚩
